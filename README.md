# anime_feature_extraction
Feature engineering and exploratory data analysis on anime dataset using Pandas &amp; NumPy

🎬 Anime Feature Engineering & Data Analysis
📌 Overview

This project focuses on extracting meaningful features from an anime dataset and performing exploratory data analysis (EDA) using Python.
The goal is to transform raw data into structured insights that can be useful for further analysis or machine learning tasks.

🎯 Objectives

Clean and preprocess raw dataset

Extract useful features from complex columns

Convert time-based data into numerical format

Perform exploratory data analysis (EDA)

Identify patterns and trends in anime data

🧰 Tech Stack

Python

Pandas

NumPy

📊 Key Operations Performed

Handling missing values

Data type conversion

Feature extraction from columns (e.g., duration, episodes)

Converting timestamp ranges into months

Sorting and filtering data

Identifying top anime based on different metrics

🔍 Key Insights

Identified longest running anime series

Analyzed trends in episode counts

Extracted structured data from raw textual formats

Observed patterns in release timelines
